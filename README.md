# Утворення відходів в Україні по галузях економіки

Аналіз даних Державної служби статистики України за 2015–2024 роки.

## Що досліджувалось
- Які галузі генерують найбільше відходів
- Динаміка змін за 10 років (вплив 2014 і 2022 років)
- Частка небезпечних відходів (клас I–III) по регіонах і галузях
- Структура відходів за матеріалами

## Головні висновки
- 91% всіх відходів — мінеральні, переважно від гірничовидобувної галузі
- Видобуток металів генерує 206 млн тонн — більше ніж усі інші галузі разом
- Дніпропетровська область — 114.6 млн тонн (48% від усіх регіонів)
- Фармацевтика має найвищу частку небезпечних відходів — 7.5% від свого обсягу
- Ринок переробки вторинної сировини в Україні — 26 млрд грн виручки у 2024 році
